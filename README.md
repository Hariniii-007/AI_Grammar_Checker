# AI Grammar Checker

## 📌 Description

AI Grammar Checker is a desktop application developed using Python and CustomTkinter. It automatically checks grammar and spelling mistakes using LanguageTool and provides corrected text with suggestions.

---

## 🚀 Features

- Grammar Correction
- Spelling Correction
- Grammar Suggestions
- Open Text File
- Save Corrected Text
- Export Report
- Copy to Clipboard
- Character Counter
- Word Counter
- Sentence Counter
- Unique Word Counter
- Reading Time
- Dark Mode
- Modern User Interface

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- LanguageTool
- Java
- NLP

---

## ▶️ Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```
